ZXID TAS3 Test Suite: Single Sign-On (SSO)

This tool allows you to run a Single Sign-On (SSO) test, testing Service Provider (SP) for TAS3 compatibility on front channel level.

Prerequisites for Test

  1. Exchange metadata of your SP with IdP

Test Input

URL of SP Start Page (URL of GUI, not the entity ID)
IdP to test against  
IdP user for test  
Password of IdP user
